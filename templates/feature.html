Detection Features

Comprehensive Phishing Detection
with Phish Guard Pro

@ Symbol Detection

Identifies URLs containing @ symbols which can be used to hide the real domain.

IP Address Detection

Detects if the URL uses an IP address instead of a domain name.

Prefix/Suffix Separation

Checks for suspicious use of hyphens separating prefix/suffix in domains.

Redirection Detection

Identifies suspicious redirection using // symbol in URLs.

Subdomain Analysis

Analyzes the number and pattern of subdomains in the URL.

URL Length Check

Detects unusually long URLs that may indicate phishing attempts.

Domain Age Analysis

Checks how long the domain has been registered (new domains are riskier).

DNS Record Check

Verifies the DNS records of the domain for suspicious configurations.

Registration Length

Analyzes domain registration duration (short registrations are riskier).

HTTPS Token Check

Verifies proper use of HTTPS tokens in the URL.

Statistical Report

Generates statistical analysis of the URL characteristics.

URL Shortening Check

Detects use of URL shortening services that may hide malicious destinations.

Web Traffic Analysis

Evaluates the website's traffic patterns for anomalies.

Dot Count Analysis

Counts the number of dots in the URL (excessive dots are suspicious).

Question Mark Count

Analyzes the number of question marks in the URL.

Ampersand Count

Counts the number of ampersands in the URL.

Equal Sign Count

Analyzes the number of equal signs in the URL.

Digit Ratio Analysis

Calculates the ratio of digits to letters in the URL.

Punycode Detection

Identifies use of Punycode encoding that may hide malicious domains.

Brand in Domain

Checks if known brands appear in the domain name (potential impersonation).

Brand in Path

Detects if known brands appear in the URL path (potential impersonation).

Suspicious TLD

Identifies use of suspicious top-level domains (TLDs).
